# Agents 135

Modular document reader for academic papers with dual-view and mindmap visualization.

## Overview

A web-based reader application designed for academic papers. Features a dual-view layout, mindmap visualization, and modular reader components. Includes a collection of papers in the `agentic_135/` directory and interactive simulators.

## Contents

- `index.html` -- Main entry point
- `reader.html` -- Document reader interface
- `mindmap.html` -- Mindmap visualization
- `reader/` -- Reader module components
- `simulators/` -- Interactive simulations
- `agentic_135/` -- Paper collection

## Getting Started

Open `index.html` in a browser to browse available papers and start reading.
