ReAct Agent Simulator

Reasoning + Acting: Interleaving Thought and Action

Concept from "Agentic Reasoning" Survey (Pages 15, 18)

ReAct (Reasoning + Acting) interleaves Chain-of-Thought reasoning with tool-use actions:

  • Thought: The agent reasons about what to do next
  • Action: The agent invokes a tool (search, calculate, lookup)
  • Observation: The agent receives the tool's output
  • Loop: Repeat until the task is solved

"The foundation of in-context agentic reasoning lies in augmenting Chain-of-Thought with the ability to take action."
