Tree of Thoughts (ToT) Simulator

Exploring Multiple Reasoning Paths with Tree Search

Concept from "Agentic Reasoning" Survey (Pages 11-12)

Tree-of-Thoughts extends Chain-of-Thought by exploring multiple reasoning paths simultaneously:

  • Thought Generation: Generate multiple candidate next-steps at each node
  • State Evaluation: Score each path's promise of reaching the goal
  • Tree Search: BFS explores breadth-first, DFS goes deep first, Beam keeps top-k
  • Backtracking: Abandon unpromising paths and try alternatives
